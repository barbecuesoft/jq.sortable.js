jq.sortable.js
==============

AppFramework jQMobi scrollable lists plugin



USAGE
--------------
Basic usage:

```html
<ul class='sortable'>
  <li>Elem1</li>
  <li>Elem2</li>
  <li>Elem3</li>
  <li>Elem4</li>
  <li>Elem5</li>
</ul>

<script type='text/javascript'>
  $(function() {
    $('.sortable').sortable();
  });
</script>
```
